# Movie Table
Live Demo — [Ознакомиться](https://bobicentr.github.io/movie-table/)

# О проекте
Приложение для ведения личной базы фильмов — что посмотреть, что уже просмотрено. Цель - работа со связкой внешнего API и собственной базы данных.

В приложении можно получать данные из базы данных и сохранять их в свою таблицу, фильтровать фильмы по нескольким параметрам. Имеется вход через панель администратора для управления записями, а также возможность предложить фильм к добавлению с гостевого аккаунта.

# Технологии
- ReactJS — основной фреймворк
- JavaScript ES6+ — логика приложения
- Supabase — база данных
- Supabase JS SDK — работа с запросами
- Bootstrap 5 — стили и адаптивная верстка
- KinopoiskUnofficial API — получение информации о фильмах

# Как установить локально

1. Склонировать репозиторий
```bash
git clone https://github.com/bobicentr/movie-table.git
cd movie-table
```

2. Установить зависимости
```bash
npm install
```

3. Запустить проект
```bash
npm run dev
```

Открыть адрес из терминала (обычно http://localhost:5173)

# Ключевые функции
- Поиск фильмов через Kinopoisk API
- Автодополнение при вводе с постерами
- Сохранение выбранных фильмов в базу данных
- Табличное отображение записей
- Фильтрация данных
- Удаление записей через панель администратора
- Возможность предложить фильм
- Адаптивный интерфейс

# Backend

- Преобразование данных API под структуру таблицы
- Работа с Supabase через SDK
